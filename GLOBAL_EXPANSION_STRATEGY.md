# AI 놀이터 글로벌 확장 전략

## 개요

본 문서는 AI 놀이터의 글로벌 시장 진출 전략을 정의합니다.
핵심 원칙: **언어 번역이 아닌, 국가별 문화 맞춤 콘텐츠 제작**

---

## 1. 타겟 국가 분석

### 1.1 선정 기준

| 기준 | 가중치 | 설명 |
|------|--------|------|
| 인터넷 보급률 | 20% | 70% 이상 필수 |
| SNS 공유 문화 | 25% | 바이럴 확산 가능성 |
| 심리테스트/운세 문화 | 25% | 기존 수요 존재 여부 |
| 결제 인프라 | 15% | 수익화 용이성 |
| 인구 규모 (15-44세) | 15% | 핵심 타겟층 크기 |

### 1.2 제외 국가 및 사유

| 국가/지역 | 제외 사유 |
|-----------|----------|
| 중국 | 만리방화벽, ICP 라이선스 필요, 검열 리스크, 별도 인프라 필요 |
| 북한 | 인터넷 접근 불가 |
| 러시아 | 규제 불확실성, 결제 제재 |
| 이란 | 국제 제재, 인터넷 제한 |
| 투르크메니스탄 | 인터넷 통제 |
| 에리트레아 | 인터넷 보급률 극히 낮음 |

| 2 | 🇹🇼 대만 | `zh-TW` | 2,400만 | 한류 친화적, 운세/심리테스트 인기, 번체 중국어권 테스트 시장 |

## 3. 국가별 문화 특성 및 콘텐츠 전략

### 3.1 Tier 1 국가 상세

#### 🇯🇵 일본 (Japanese)

**문화적 특성**
- 占い(우라나이) 문화: 혈액형, 별자리, 타로 일상화
- 診断(진단) 테스트 인기: Twitter에서 자주 바이럴
- 캐릭터 문화: 결과를 귀여운 캐릭터로 표현 선호
- 완곡한 표현: 직접적 비판보다 부드러운 조언

**콘텐츠 현지화 방향**
```
한국: "아재력 테스트"
일본: "おじさん度診断 〜あなたの中のオジサンは何歳?〜"
      (아저씨 지수 진단 ~당신 안의 아저씨는 몇 살?~)

한국: "MBTI 테스트"  
일본: "性格タイプ診断 〜16の性格であなたを分析〜"
      (성격 타입 진단 ~16가지 성격으로 당신을 분석~)
```

**인기 예상 테스트 유형**
1. 혈액형 × 성격 조합
2. 애니메이션/만화 캐릭터 매칭
3. 일본 사계절/지역 성격
4. 会社(회사) 생활 관련 (社畜 테스트 등)
5. 恋愛(연애) 스타일 진단

**플랫폼 전략**
- Twitter (X): 메인 바이럴 채널
- LINE: 공유 및 알림
- Instagram: 결과 이미지 공유

#### 🇹🇼 대만 (Traditional Chinese)

**문화적 특성**
- 星座(별자리) 운세 매우 인기
- 心理測驗(심리테스트) 검색량 높음
- 한국 콘텐츠에 대한 높은 수용도
- 可愛(귀여움) 문화 선호

**콘텐츠 현지화 방향**
```

      (당신의 연애 매력은 어디에? 당신의 매력 유형을 테스트!)

한국: "전생 테스트"
대만: "你的前世是什麼？探索靈魂的記憶"
      (당신의 전생은 무엇? 영혼의 기억을 탐색)
```

**인기 예상 테스트 유형**
1. 별자리 기반 성격/운세
2. 台灣特色 (대만 특색) - 야시장 음식, 지역 성격
3. 연예인/아이돌 매칭
4. 직장 생활 (上班族 테스트)
5. 연애/궁합 테스트

**플랫폼 전략**
- Facebook: 여전히 강세
- Instagram: 젊은 층
- LINE: 대만에서도 메인 메신저
- PTT/Dcard: 커뮤니티 바이럴

#### 🇹🇭 태국 (Thai)

**문화적 특성**
- ดูดวง(두둥): 점술/운세 일상의 일부
- 불교 문화: 전생, 업보 개념 친숙
- สนุก(사눅): 재미 추구 문화 강함

```
한국: "번아웃 지수 테스트"
태국: "คุณเหนื่อยล้าแค่ไหน? ทดสอบระดับ Burnout"
```

**인기 예상 테스트 유형**
1. ดวงความรัก (연애운) - 태국 점술 스타일
2. 전생/업보 관련 테스트
3. 태국 드라마 캐릭터 매칭
4. 띠별 성격 (태국식 12간지)
5. 태국 음식/지역 성격

**플랫폼 전략**
- Facebook: 메인 플랫폼
- LINE: 태국 국민 메신저
- TikTok: 젊은 층 바이럴
- Twitter: 트렌드 확산
